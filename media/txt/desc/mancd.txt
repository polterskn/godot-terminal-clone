----------------------------------------------------------------
[center]Change directory[/center]
----------------------------------------------------------------
The command line directive cd stands for change directory, and
is used for navigation in the directory tree.
You can only navigate to directories that are within the current
directory. To show directories you can access use the ls command.
----------------------------------------------------------------
Syntax:					cd /[directory]

Example of use:			cd /myStuff
----------------------------------------------------------------
[center]Basic navigation commands[/center]
----------------------------------------------------------------
	cd .. <------------- Navigate to parent (previous) directory.
	cd / <-------------- Navigate to root (/home/user) directory.
